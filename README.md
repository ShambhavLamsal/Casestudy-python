# Heart Disease Data Visualization – Case Study

## Overview

This project explores the **Heart Disease UCI dataset** using Python to analyze patterns related to heart disease.
The goal of the case study is to understand relationships between different health variables using data visualization techniques.

## Dataset

The dataset used in this project is the **Heart Disease UCI dataset**, which contains medical attributes such as:

* Age
* Cholesterol level
* Maximum heart rate
* Chest pain type
* Presence of heart disease

## Tools and Libraries

The following Python libraries were used:

* Pandas
* Matplotlib
* Seaborn
* Plotly

## Visualizations Created

Several visualizations were created to explore the dataset:

* Scatter Plot – Age vs Maximum Heart Rate
* Histogram – Cholesterol Distribution
* Box Plot – Cholesterol vs Heart Disease
* Heatmap – Correlation between variables
* Line Chart
* Subplots comparing Age and Cholesterol

## Project Files

* `case_study.ipynb` – Jupyter Notebook containing the analysis
* `case_study.html` – Exported HTML version
* `case_study.pdf` – PDF version of the case study
* `heart_disease_uci.csv` – Dataset used in the analysis

## Objective

The objective of this project is to demonstrate **data visualization techniques and exploratory data analysis using Python**.
